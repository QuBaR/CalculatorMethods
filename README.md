# CalculatorMethods

A simple console-based calculator application written in C# (.NET 8). The program allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers (Note: currently, division performs multiplication—see "Known Issues")

## Requirements

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- C# 12.0 compatible environment (e.g., Visual Studio 2022)

1. Build and run the application.
